Multi item sorter made smaller by ignoring the silent requirement.
## Features
- Sorts items at hopperspeed with a 40gt delay between item types.
- Can sort up to 54 items per slice
- Can sort 64 and 16 stackable items
- Accepts (but can't sort) unstackables
- 2x6x6 size
- Can be fully hopperlocked
## Notes
Dispenser with powdered snow bucket can be replaced with a water bucket and waterloggable block. [Video on the design](<https://youtu.be/LDBwWtsDjxY?si=9cSSiQ_-JHF2IqPZ>) with a block by block tutorial
