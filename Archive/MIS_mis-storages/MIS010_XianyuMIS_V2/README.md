Cart-based MIS storage for intermediate players. Pre-made decoration available.
## Features
- 42 MIS categories
  - Whitelisters are easily accessible to the player.
- 41 bulk items.
  - Loads loose bulk items into boxes at 2x hopperspeed, box sorter runs at 24gt.
- Unstackable sorter.
- Safety features:
  - If the system runs too low on blockers, it will sound an alarm and stop the system.
  - Emergency shut off lever that stops the system; important to note that this lever should not be treated as a pause because it leads to items despawning.
  - Indicator lights including overflow indicators and an indicator for the unstackable sorter needing more arrows.
- 1813 hoppers
## Notes
Most inventories have signs indicating how to prefill them.
