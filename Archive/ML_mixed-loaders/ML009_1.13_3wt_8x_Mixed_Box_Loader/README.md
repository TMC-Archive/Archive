**Original post:** [View on Discord](https://discord.com/channels/1375556143186837695/1469044024110350601)

Loads single or mixed item types into shulker boxes at 8x hopper speed (approximately 72k/hr)
## Features
- Small (5x7x3)
- No item spitting with random input
- Locally clocked
- Outputs boxes into a global hopperline
## Considerations
- No ice blocks in the item stream, making it not that Nether friendly
- Can buffer items inside the droppers
## Notes
For Minecraft 1.13 compatibility, use droppers instead of composters.
