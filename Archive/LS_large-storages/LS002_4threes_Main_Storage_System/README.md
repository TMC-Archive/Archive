An advanced Minecraft storage system created by 4three, with components sourced from storage tech archives.
## Features
- 864 chest hall items, two double chest capacity per item. 8 chest per slice UI.
- 216 bulk items, 1,5 million capacity per item. Quad display UI.
- 10 multi item categories
- Unstackables sorter
- Empty box crafter automatically supplying the whole system with boxes
- Floor box system for empty boxes
- Parallel item unloading
- Multiple peripherals (192 furnace array, crafting station, restock station, anvil replacer)
- Basic error checking (and automatic input locking)
- Includes chunk loading grid and item layout
- 96,4% Hopper locking
## Instructions
- Make sure to align chunkloaders in the nether side to match the overworld and link them correctly.
